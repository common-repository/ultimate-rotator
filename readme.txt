== Description ==
Ultimate Rotator helps you to create rotating images and sliding contents. It has some widgets and shortcodes to display rotator items. You can integrate the rotator in your sidebar, post, page or anywhere else in your template. 

Features

  * Rotator items can be images, videos or other html contents
  * Support vertical and horizontal slide contents
  * Support multiple items per cycle of the slide
  * Support multiple rotators in one page.
  * Control the dimension of the slide
  * Built-in widgets & shortcodes for display
  * Control the slide timeout, speed and more
  * Support description for each slide item

Read more details about this <a href="http://gidd.org/wordpress-plugins/ultimate-rotator-free-image-banner-rotating-plugin/" target="_blank">rotator plugin here</a>, including demo site & how to use.


== Installation ==
1.  Upload Ultimate Rotator to the /wp-content/plugins/ directory.
2.  Activate the plugin through the Plugins menu item in the WordPress Dashboard.
3.  Once the plugin is activated, add your rotator items and use widgets or shortcodes to display.
